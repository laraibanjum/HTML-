Here we will learn about some the Basic Git Command.

Let's Start

  • git init

    Starts a new Git repository in your current folder (only needs to be done once).

  • git status

    Shows what’s been changed, what’s ready to be saved (staged), and what’s not.

  • git add .

    Adds all your changes (files/folders) to be saved in the next commit.

  • git add filename

    Adds just one specific file to be saved in the next commit.

  • git commit -m "message"

    Saves a snapshot of your code with a message explaining what changed.

  • git log

    Shows a history of all your commits.

  • git branch

    Lists all branches in your repo (used for working on different versions).

  • git checkout -b branch-name

    Creates a new branch and switches to it.

  • git merge branch-name

    Merges the changes from one branch into your current one.

  • git push -u origin main

    Sends your commits to GitHub (or another remote).

  • git pull

    Downloads the latest changes from GitHub to your local machine.

  • git clone [URL]

    Copies a repo from GitHub to your computer.
